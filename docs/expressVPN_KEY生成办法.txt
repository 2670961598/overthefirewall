

一、创建一个临时邮箱，在手机上注册ExpressVPN 输入临时邮箱地址，（我是用的模拟器在电脑上面操作）


     临时邮箱生成网站，亲测均可

     https://10minutemail.org/

     http://24mail.chacuo.net/

    第一步，不需要开梯子，内网的情况下可以操作。你临时邮箱会收到一个邮件，里面有你的验证码。


二、打开ExpressVPN，点击上面的连接（此时开启梯子才能连下面的链接）。https://www.expressvpn.com/
     
    点击进入 My account 

    点击下面 Forgot you password   忘记你的密码  再输入邮箱地址（一次性的邮箱）
     
    点击 send instructions      发送指令
    
三、打开邮箱，收到第二封邮件打开，点击 setyoupassword，设置密码。 
    
     点击setpassword ，设置密码 按钮
     
      A \系统自动生成一个新的密码复制并保存    Continue with this password  

     
      B\ createmy  opnpasswor 创建新密码       Crate my own password

四、有了密码，在https://www.expressvpn.com/  上面输入邮箱及密码 就能收到新的注册KEY，你就可以输到PC

客户端了
